# AI-Powered Sentiment Analysis Web Application

AI-based sentiment analysis web app built using Streamlit and LLaMA (via Ollama) to classify text into Positive, Negative, Neutral, or Mixed with confidence scores.

## Features
- Single text sentiment analysis
- Batch analysis with CSV/TXT files
- Interactive chatbot with memory
- Confidence-based filtering
- WordCloud visualization

## Technologies
Python, Streamlit, LangChain, Ollama (LLaMA 3.2), Pandas, Matplotlib
